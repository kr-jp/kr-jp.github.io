# 文章は「時間」が書く
あなたが天才でもない限り、数日で長文、かつ、論理的な構造の文章を書き上げることはできません。

卒業論文は、時間管理をすれば、誰でも書けます。「言わなくても知ってるよ！」と思うでしょ？でもやらない人がいるんですよ。

「バランスの取れた食事をしましょう」「定期的に運動をしましょう」「脂っこい食べ物や塩分は控えましょう」「アルコールは適量を守って飲みましょう」「タバコはやめましょう」と、いくら言っても聞く耳を持たない。道理で儲かる医者がいるわけだ。

「ショート動画は中毒になりますから本を読みましょう」「自分と他人を比べることでうつ病になる可能性がありますから、SNS中毒にならないようにしましょう」といくら言っても、聞かない人がいる。

「時間管理をちゃんとすれば、普通の卒論はみんな間違いなく書けるから」と、いくら言っても聞いてくれない学生がいるので、なぜそれが必要なのか、ここで触れておきたいと思います。
